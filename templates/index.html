Welcome to our Flask web application!

Our application provides the following functionalities:

  • Adding new user to database: You can add new users to our database by filling out a simple form. The form collects information such as the user's name and email. Once the form is submitted, the user's information is stored in the database.
  • Displaying existing users as a list: You can view a list of all the existing users in our database. The list displays each user's name and email.
  • Viewing embedded image and gif media: Our application supports the embedding of image and gif media. You can view the embedded media by navigating to the "Media" section of our application.
